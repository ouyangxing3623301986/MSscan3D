MSscan3D 官方用户手册

作者：欧阳兴 更新时间：2026年5月 适用版本：v3.3 系列

这份手册是给第一次搭建结构光三维扫描仪的用户看的。阅读顺序建议按“安装软件 -> 连接硬件 -> 标定 -> 扫描 -> 对齐 -> 导出”走，不需要先理解所有算法名词。

硬件插件化支持 HDMI 投影仪、工业 DLP 光机、海康/迈德威视工业相机和旋转台插件。
标定决定精度相机、投影仪、标定板和扫描物体都必须稳定，标定误差合格后再扫描。
面向实用导出扫描结果可用于 3D 打印、逆向建模、尺寸参考和教学演示。

推荐工作流程

1

安装主程序。安装包内置常用驱动和条纹资源，先不要急着接复杂设备。

2

连接设备。两台工业相机分别接主板 USB 3.0，投影仪走 HDMI 扩展屏，旋转台单独接 USB/串口。

3

完成标定。拍摄多组清晰标定板图像，确认重投影误差和标定板 3D 形状正常。

4

采集点云。先扫一个简单物体，再使用转台做 6 到 12 个视角的 360 度采集。

5

处理导出。对齐、合并、清理杂点，最后导出 STL、OBJ 或 PLY。

先扫简单物体

第一次测试请使用不反光、表面有形状特征的小物件。透明、黑亮、镜面金属、毛发和纯平大白面都不适合拿来做第一次验证。

电脑配置与软件安装

安装准备 Windows 10/11 64 位

MSscan3D 的计算压力主要来自相机图像采集、相位计算、点云显示和多视角拼接。电脑配置越好，实时预览和多组点云处理越顺。

项目最低建议推荐配置
系统Windows 10 64 位Windows 11 64 位
CPU4 核处理器6 核或更高
内存8 GB16 GB 或 32 GB
显卡支持 DirectX 的独立或核显NVIDIA/AMD 独显，显存 2 GB 以上
接口USB 3.0、HDMI多个主板原生 USB 3.0、独立 HDMI 输出
硬盘10 GB 可用空间SSD，预留 50 GB 以上扫描项目空间

安装步骤

  1. 从官网首页点击“立即免费下载”下载安装包。
  2. 右键安装包，选择“以管理员身份运行”。
  3. 使用默认安装路径，驱动、条纹和示例配置会一起安装。
  4. 安装完成后重启电脑，再连接相机、投影仪和旋转台。
  5. 第一次启动软件时，先打开设备面板确认各插件状态。

不要混装多个相机驱动版本

如果电脑里已经装过其他工业相机 SDK，建议先确认版本兼容。驱动冲突时常见表现是相机能在厂家软件里打开，但 MSscan3D 里无法稳定取图。

相机、投影仪与旋转台接线

硬件拓扑 USB 3.0 优先

硬件连接的原则很简单：相机要稳定高速传图，投影仪要保持原始分辨率输出，旋转台要单独供电并能被软件识别。

接线检查表

  • 相机不要接到廉价 USB HUB。两台相机同时传图时，带宽不足会导致丢帧或卡死。
  • 投影仪在 Windows 中设置为“扩展这些显示器”，不要使用复制屏幕。
  • 投影仪缩放比例保持 100%，分辨率使用设备原生分辨率。
  • 旋转台先接电源，再接控制线，最后在软件里刷新设备。
  • 所有支架、相机、镜头、投影仪固定后再标定，标定后不要再移动它们。

投影模式与曝光设置

条纹投影 曝光调节

MSscan3D 支持两类常见投影方式：普通 HDMI 视频投影和工业 DLP 硬触发投影。普通用户优先使用 HDMI；需要高速同步采集时再使用硬触发光机。

模式适合场景注意事项
HDMI 扩展屏桌面扫描、教学、DIY 套件必须关闭系统缩放和投影仪图像增强。
工业 DLP 硬触发高速采集、生产线、固定工位需要确认触发线、电平、曝光和图案间隔。

曝光调节方法

  1. 打开相机实时预览，投影白图。
  2. 调镜头光圈，让白色区域亮但不过曝。
  3. 调曝光时间，让条纹边缘清晰、噪点可控。
  4. 切换黑图、白图和正弦条纹，确认画面不会一片黑或一片白。
  5. 保存曝光参数到当前设备配置。

不要开投影仪自动梯形校正

自动梯形校正会把软件输出的条纹重新拉伸，导致相位计算不准确。请在物理上摆正投影仪，而不是依赖投影仪内部算法修图。

标定板选择与拍摄姿态

标定 精度基础

标定的作用是把相机图像里的二维像素转换成真实三维坐标。标定做不好，后面扫描再多次也只是得到“看起来像”的模型，尺寸和形状都会漂。

标定板怎么选

  • 小物件：使用小格距标定板，保证标定板能覆盖主要扫描视场。
  • 中等物件：使用中等格距标定板，拍摄时覆盖近、中、远位置。
  • 大物件：使用更大标定板或扩大相机基线，同时接受分辨率降低。

推荐拍摄姿态

A

正对相机拍 1 到 2 张，用于确认画面清晰和角点识别稳定。

B

向左、向右、向上、向下倾斜各拍几张，让标定覆盖不同深度。

C

近距离和远距离各拍几张，覆盖真实扫描物体将要出现的空间范围。

D

每张图都要完整看到标定板关键区域，边缘不要严重虚焦或过曝。

拍得多不等于好

建议保留 12 到 20 张质量好的标定图。模糊、反光、角点识别错的图宁可删除，不要混进标定计算。

重投影误差确认与返工判断

质量检查 返工依据

重投影误差可以理解成“软件根据标定结果反推角点位置时，和真实图像角点差了多少”。数值越小，说明标定越稳定，但还要结合实际扫描形状判断。

现象可能原因处理方式
误差小，但扫描板子弯曲标定姿态覆盖不够，或者投影缩放被改动检查投影分辨率，重新拍近远和倾斜姿态。
角点识别很慢曝光不均、背景反差太低、板子反光换深色背景，降低曝光，清洁标定板。
左右相机结果差异大一台相机虚焦或接口掉帧分别查看实时画面，检查镜头和 USB 口。
扫描尺寸偏大或偏小标定板格距填写错误核对标定板实际规格，重新建标定配置。

通过标定板实扫确认

  1. 切到扫描项目，直接扫描标定板。
  2. 查看生成的 3D 面是否基本平直。
  3. 用测量工具检查格距或板上已知长度。
  4. 如果平面明显拱起、扭曲或尺寸不对，回到标定环节返工。

新建项目与设置扫描体积

项目管理 扫描范围

每个扫描对象建议单独建一个项目。这样相机参数、转台参数、原始点云、对齐结果和导出文件都能放在同一个目录里，后面好查找。

项目命名建议

D:\MSscan3D_Project\
  2026-05-25_gear_test\
    raw_scans\
    aligned\
    mesh\
    export\

扫描体积和裁切面

扫描体积就是你希望软件保留的空间范围。设置合适的裁切面可以去掉桌面、转台底盘和背景杂点，减少后续清理工作。

  • 物体尽量放在相机共同视场中心。
  • 转台上的物体不要超过标定覆盖区域。
  • 如果桌面和物体颜色接近，建议设置底部裁切面。
  • 要做尺寸比较时，项目中不要随意缩放点云。

旋转台 360 度自动扫描

转台 多视角采集

旋转台适合扫描小到中等大小的静态物体。它的好处是物体不动、相机不动，只让物体按固定角度旋转，后续对齐更稳定。

推荐参数

目标建议视角数每次角度说明
快速预览6 次60°适合先看整体是否能扫出来。
常规建模8 次45°速度和完整度比较均衡。
复杂零件12 次30°遮挡多时使用，但数据量更大。

操作步骤

  1. 在设备面板确认旋转台插件已连接。
  2. 把物体放在转台中心，避免旋转时晃动。
  3. 设置视角数、旋转方向、停稳等待时间。
  4. 点击自动扫描，观察每个角度是否正常生成点云。
  5. 扫描完后先保存项目，再做自动对齐和合并。

物体必须稳

如果物体在转台上滑动或倾倒，对齐会失败。底部可以用少量双面胶或橡皮泥固定，但不要遮挡需要扫描的关键面。

点云对齐、合并与清理

点云处理 清理杂点

单次扫描只能看到物体正面对相机的一部分。要得到完整模型，需要把多个视角的数据对齐到同一个坐标系，再合并成一个整体。

对齐方式

方式适合对象要点
转台角度对齐使用旋转台扫描的物体先完成转轴校准，角度记录要准确。
几何自动对齐表面有足够特征的零件相邻视角要有重叠区域。
手动三点对齐自动对齐失败的局部数据三点应分散，避免都选在一条线上。
标记点对齐大物件或特征少的物体标记点要在多个视角中都能看到。

清理顺序

  1. 先删掉明显的桌面、支架和空气杂点。
  2. 再做对齐，不要一开始就大幅平滑。
  3. 对齐完成后检查接缝、错层和重复面。
  4. 最后再合并、补洞、平滑和降面。

导入导出格式

导出 网格

导出前请先确认模型方向、尺寸单位和坐标原点。不同下游软件对格式的要求不同，选错格式会导致颜色丢失、比例错误或无法导入。

格式用途说明
STL3D 打印、切片软件只保存几何，不保存颜色纹理。
OBJ逆向建模、贴图模型可配合材质和纹理文件保存颜色。
PLY点云处理、科研软件适合保存点云或带颜色顶点。
ASC/XYZ坐标点导出适合自己写程序分析点坐标。

导出前检查

  • 模型是否已经合并成需要的整体。
  • 底面是否已经贴正，坐标轴方向是否符合打印或 CAD 软件要求。
  • 是否需要保留颜色。需要颜色时优先 OBJ 或 PLY。
  • 是否需要降面。过大的 STL 会让切片软件打开很慢。

脚本与自动化接口

自动化 XML 指令

MSscan3D 提供脚本化控制能力，用于重复扫描、固定工位采集、转台自动化和二次开发。普通用户可以先从单条命令测试开始，不需要一上来写完整程序。

常用指令示例

指令作用
<ConnectDevices />连接相机、投影仪和旋转台。
<SetExposure Value="22000" />设置相机曝光时间，单位微秒。
<CaptureFrame SaveAs="Scan_001" />执行一次条纹投影、拍照和点云计算。
<RotateTurntable Angle="45" />控制旋转台转动指定角度。
<ExportModel Format="STL" Path="D:\out\part.stl" />导出模型文件。

8 视角自动扫描示例

<MSscanScript>
  <ConnectDevices />
  <SetExposure Value="22000" />

  <Loop Iterations="8">
    <CaptureFrame SaveAs="RotaryScan" />
    <AutoRegister Mode="Rotary" />
    <RotateTurntable Angle="45" />
  </Loop>

  <MergeScans />
  <ExportModel Format="STL" Path="D:\MSscan_Out\part_complete.stl" />
</MSscanScript>

先单步测试

自动化脚本正式运行前，建议先单独测试连接、曝光、采集和转台旋转。每一步都稳定后，再组合成完整流程。

问题优先检查处理建议
